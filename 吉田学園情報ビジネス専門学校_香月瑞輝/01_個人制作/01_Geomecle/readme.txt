◆ 作品について ◆

◇ 使用言語 : C++
◇ 制作環境 : VisualStudio2015、DirectX9
◇ 開発期間 : 2022/7 ～ 2022/9 (三か月)
◇ ジャンル : シューティング

◆ 作品説明 ◆
全方位2Dシューティングです。
超大量のポリゴンによるパーテイクルで爽快感とインパクトを味わおう！

◆ 動作確認済み環境 ◆
Windows10、Windows11
（ XInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆ 操作方法 ◆

◆ キーボード

◇ 移動 : WASD
◇ 弾   : ↑←↓→
◇ ボム : SPACE

◆ ゲームパッド

◇ 移動 : 左スティック
◇ 弾   : 右スティック
◇ ボム : L / R
